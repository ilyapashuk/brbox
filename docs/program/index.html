о программе

утилита brbox - это написанное на языке Golang консольное приложение.

конфигурирование

все файлы данных, используемые приложением в процессе работы, должны находиться в каталоге config рядом с исполняемым файлом приложения.

подробнее о синтаксисе конфигурационных файлов

приложение реализует несколько субкоманд:

  • prepair
    осуществляет конвертацию текстового файла к формату bxt, автоматизируя некоторые операции корректуры текста
  • conv осуществляет конвертацию между brf, брайлевским юникодом и bxt во всех направлениях, а также может применять к данным некоторые преобразования, называемые фильтрами, в процессе конвертации
  • tableonly обрабатывает файл bxt, заменяя символом брайлевского шеститочия все символы, неизвестные таблице. аналогично последнему этапу обработки командой prepair при исходных настройках. выведено в отдельную команду для удобства пользователей.

общие правила

в большинстве случаев вместо имён входного и выходного файлов можно писать символ дефиса (-), что вызовит получение входных данных из stdin или вывод их в stdout, однако в этом случае может потребоваться указание типа данных в дополнительном параметре, поскольку приложение не сможет определить его по расширению файла.

когда на вход поступает текстовый файл, то, когда он содержит bom, он считывается как utf8, иначе он считывается в кодировке ansi, для русского языка cp1251. поддерживаются переводы строк в стиле windows и unix.

когда программа в результате своей работы генерирует текстовый файл, то он всегда выводится как utf8 с bom. если приложение выполняется в среде Windows, то концы строк тоже будут в стиле Windows, иначе будет использован стиль unix.

когда работа идёт со stdin/stdout, всегда выводится и ожидается на вход utf8 без bom с концами строк в стиле unix.
